Show realtime server and execution environment information.

== Description ==
Did You ever needed to show some realtime server and execution environment information?
'Host Info' will provide you more information on the webserver that´s hosting your Wordpress site.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/` directory, or install plugin through the WordPress 'Plugins' screen directly
2. Activate plugin via 'Plugins' screen in WordPress
3. Go to 'Tools' and select 'Host Info'

== Frequently Asked Questions ==
= How to see server info? =

1. Install the plugin files
2. Activate plugin
3. In the Admin-menu, at 'Tools' look for: **[Host Info]**

= Why can I not see Uptime and so on? =

Sorry, my guess is that you are using a Windows server. I´ve only tested it on Linux servers..
Feel free to offer me a live Windows server to develop this feature for you!

= When will there be a translation to my language? =

You are welcome to translate this plugin using POEDIT. Provide me with the translated files and I´ll put them online for you.
